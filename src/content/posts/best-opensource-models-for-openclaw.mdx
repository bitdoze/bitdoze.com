If you run OpenClaw, you need a powerful LLM behind it. Open source models offer comparable performance at a fraction of cost.

## Quick Comparison

GLM-4: 131K context, 12B params, $0.07/M
MiniMax: 1.05M context, 2B params, $0.50/M
Claude: 200K context, 200B params, $15/M

## GLM-4

GLM-4 is a massive open source model with 131 billion parameters.

Massive context window for code reviews
Multi-step reasoning for complex tasks
Excellent coding performance
Incredible value at 5% of Claude cost

### Pricing

API: $0.50 per million tokens

### Usage

Visit https://zhipu.ai/glm-4/ to create account.

### Configuration

openclaw configure --section models
Set model to zhipu/glm-4

## MiniMax

MiniMax uses Mixture-of-Experts architecture for incredible efficiency.

Superior at multi-step reasoning
Strong at following instructions
Very good performance for parameter budget

### Pricing

API: $0.10 per million tokens

### Usage

Visit https://together.ai/ to create account.

### Configuration

openclaw configure --section models
Set model to together-ai/mini-max-3.5-turbo

## Comparison to Claude

GLM-4 matches or exceeds Claude on most tasks.
